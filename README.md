# clean-cache

`go` 写的清理 `node` 项目缓存，超级无敌快

<br />

## Usage

### install

```shell
go install github.com/markthree/clean-cache
```

### clean

```shell
clean-cache
```

#### 移除 dist 和 .output

```shell
clean-cache -d
```

#### 移除 node_modules

```shell
clean-cache -n
```
